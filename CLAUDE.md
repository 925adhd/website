# 925 ADHD Website

## Stack
- Next.js 14+ (App Router, static export)
- TypeScript (strict)
- Tailwind CSS v4
- motion/react for animations
- lucide-react for icons
- Web3Forms for contact form (placeholder key)
- Deployed on GitHub Pages / Vercel

## Structure
- app/ — pages and layouts
- src/components/ — reusable UI components (Navbar, Footer)
- src/lib/ — blog data and content
- public/ — static assets (images, sitemap, robots.txt)
- _old_site/ — original static HTML site (gitignored)

## Domain
- Live: 925adhd.com

## Conventions
- Mobile-first (default styles = mobile, md: = desktop)
- All images WebP with alt, width, height
- Text contrast minimum: 65% opacity
- Semantic HTML: main, section, article, nav, footer
- One H1 per page, never skip heading levels
- page.tsx = server component (metadata), *-page.tsx = client component (content)
- Brand colors: brand-primary (#0f172a), brand-accent (#7c3aed), brand-green (#22c55e)
